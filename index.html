About me

I'm a sophomore at Brown University who's interested in both computer science and graphic design. Beginning with Visual Basic in high school, I quickly became fascinated with writing programs that produced graphics. I picked up HTML, CSS and JavaScript, creating my first website back in 2013. This also led me to working on a web application with the company Riperian. In addion to web languages, I've also been learning Java since high school. In Software Engineering and 2D Game Engines classes at Brown, I got to work with a Java-based game engine and then make my own. Other interests include Chinese, Karate (Okinawan Goju-ryu), making music, and video editing.
