RAO

Riperian Account Opening

Also known as Riperian Account Opening, this web application allows for flexible, extensible creation of account opening forms without needing to know any programming languages. AngularJS allows the app to dynamically load interactive elements, and a customized Bootstrap theme (which I made based on Google's material design) provides a coherent interface. The app was developed entirely by myself.
