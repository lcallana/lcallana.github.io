Minecraft-Sweeper (2013)

Minecraft-Sweeper is a program based on Minecraft and Minesweeper. Made with the help of my Senior Project mentor, Ben LeVeque, this project was designed as a precursor to development on my original website. Building this game taught me the basics of HTML, CSS, and Javascript, none of which I was familiar with at the time.
