RAO (2015)

Also known as Riperian Account Opening, this web application allows for flexible, extensible creation of account opening forms. It's currently being developed for Riperian entirely by myself, and it uses some external plugins such as Bootstrap and JQuery Mask Plugin. The form generator allows a user to make a form without needing to know any programming languages.

The form shown below is a prototype of a bank account opening system. An interactive demo should be available soon.
