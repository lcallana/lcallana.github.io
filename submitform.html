Riperian Account Opening (2015)
A responsive object-oriented approach to web forms

This web application allows for flexible, extensible creation of account opening forms. It was developed for Riperian entirely by myself (and using some external plugins such as Bootstrap and JQuery Mask Plugin). This form generator allows a user (who may not know too much javascript) to make a form without needing to know any programming languages.

The form shown below is a prototype of a bank account opening system. An interactive demo will be available soon.
